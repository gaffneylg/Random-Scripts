# Arbitrary

Dump repo to push up any files needed. Mostly used for setup scripts or config related files.

<br />

### Terminal colours: 
Mac (iTerm, VS Code etc):
- Background: #0a3c46
- Foreground: #0aa032

VS Code on Windows:
```
    "workbench.colorCustomizations": {
        "terminal.background": "#0d3b38",
        "terminal.foreground": "#45b354"
    },
```

Git terminal colours to be used in line with the above:
```
[color]
	interactive = auto
	ui = true
	pager = true

[color "status"]
	added = yellow
	changed = red bold
	untracked = magenta bold

[color "branch"]
	current = yellow

[color "diff"]
	meta = white
	new = yellow
	old = red bold
```
